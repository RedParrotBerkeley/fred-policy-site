# fred-policy-site

Static policy site for Fred Messaging, including public privacy, terms, and consent pages.

## Purpose

This repository hosts a minimal static website used to publish messaging policy documents:

- Privacy Policy
- Terms and Conditions
- Messaging Consent

## Files

- `index.html` — landing page linking to policy documents
- `privacy.html` — privacy policy
- `terms.html` — terms and conditions
- `messaging-consent.html` — consent language for messaging workflows

## Deployment

This repo is designed for static hosting (e.g., GitHub Pages, Netlify, Cloudflare Pages, or any basic web server).

## Status

Active as a lightweight policy/documentation site.
